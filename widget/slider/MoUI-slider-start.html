Getting Started

Include the dependencies: jQuery, MoUI slider CSS + Javascript.


<link href="MoUI-slider.css" rel="stylesheet">
<script src="jquery.js"></script>
<script src="MoUI-slider.js"></script>

Add your slider html.


<div class="slider">
  <ul>
    <li style="background-image: url('img/sunset.jpg');"></li>
    <li style="background-image: url('img/wood.jpg');"></li>
    <li style="background-image: url('img/subway.jpg');"></li>
    <li style="background-image: url('img/shop.jpg');"></li>    
  </ul>
</div>

Initialize MoUI slider.

$('.slider').MoUISlider();
