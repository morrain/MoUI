Getting Started

Include the dependencies: jQuery, Bootstrap and MoUI Switch CSS + Javascript.


<link href="bootstrap.css" rel="stylesheet">
<link href="MoUI-switch.css" rel="stylesheet">
<script src="jquery.js"></script>
<script src="MoUI-switch.js"></script>

Add your checkbox.

<input type="checkbox" name="my-checkbox" checked>

Initialize MoUI Switch.

$("[name='my-checkbox']").MoUISwitch();
